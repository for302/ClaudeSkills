# PDF Contents Extractor (페이지별 분석)

PDF 파일에서 교육과정 정보를 추출하여 구조화된 마크다운 파일을 생성하는 스킬입니다.
컨텍스트 효율을 위해 페이지별로 분석합니다.

## Purpose

이 스킬은 다음 작업을 수행합니다:
- PDF 파일을 페이지별 PNG 이미지로 분리
- 각 페이지를 개별적으로 분석하여 교육과정 정보 추출
- 교육과정 제목, 커리큘럼 등을 마크다운으로 정리
- 컨텍스트를 효율적으로 관리 (Sub Agent 활용)

## When to Use This Skill

다음 상황에서 이 스킬을 활성화하세요:
- 교육 커리큘럼 PDF를 분석할 때
- 많은 페이지의 PDF를 처리해야 할 때
- 표나 복잡한 레이아웃이 있는 PDF를 분석할 때
- 컨텍스트 제한이 걱정될 때

## Workflow

### Step 1: PDF를 페이지별 이미지로 분리

```bash
python split_pdf.py <pdf_path>
```

### Step 2: Sub Agent로 자동 분석

각 페이지를 병렬로 Sub Agent에게 분석 요청합니다.
Sub Agent는 독립적인 컨텍스트에서 실행되므로 /clear가 필요 없습니다.

### Step 3: 결과 취합

분석된 내용을 순서대로 .md 파일에 저장합니다.

## Output Format (2가지 유형)

### 유형 1: 교육과정 체계도/목록 (제목만 있는 경우)

교육과정 제목만 나열된 체계도 페이지에서 추출:

```markdown
## [분야명]

- [기본] [과정코드] 과정제목1
- [응용] [과정코드] 과정제목2
- [심화] [과정코드] 과정제목3

---
```

예시:
```markdown
## AI+경영

- [기본] [AI+] ChatGPT 마케팅 활용 실무
- [응용] [AI+] 데이터 기반 인사관리, People Analytics
- [기본] [AI+] ChatGPT를 활용한 인사관리 실무
- [응용] [AI+] AI 경영혁신 리더십
- [기본] [AI+] AI 업무혁신 리더십
- [심화] [AI+] 변화를 리드하는 AI 혁신 리더십
- [심화] [AI+] AX 기반 비즈니스 혁신 로드맵 구축
- [심화] [AI+] 디자인씽킹 기반 AX 사업전환 혁신
- [응용] [AI+] 생성형 AI를 활용한 나만의 커리어 로드맵 설계
- [기본] [AI+] 법률 AI 활용 A to Z

---
```

### 유형 2: 상세 교육과정 (커리큘럼이 있는 경우)

```markdown
## [교육과정 제목]

**카테고리**: [분류]
**교육기간**: [기간]
**교육비**: [비용]

### 커리큘럼
- [내용1]
- [내용2]

---
```

## Sub Agent Prompt Template

```
페이지 이미지를 분석하여 교육과정 정보를 추출하세요.

**추출 우선순위:**
1. 교육과정 제목이 있으면 반드시 추출 (상세 정보 없어도 OK)
2. 체계도/로드맵 형태면 분야별로 과정 제목 목록 추출
3. 상세 커리큘럼이 있으면 함께 추출

**출력 형식:**

[체계도/목록인 경우]
## [분야명]
- [수준] [과정코드] 과정제목
...

[상세 과정인 경우]
## [교육과정 제목]
**카테고리**: ...
**교육기간**: ...
### 커리큘럼
- ...

[교육과정 정보가 전혀 없는 경우만]
SKIP: [이유]
```

## SKIP 기준 (엄격하게 적용)

다음 경우에만 SKIP:
- 표지, 목차 페이지
- 연락처, 위치 안내 페이지
- 교육과정 제목이 전혀 없는 순수 안내 페이지

다음은 SKIP하면 안됨:
- 교육과정 체계도/로드맵 (과정 제목 있음)
- 과정 목록표 (과정 제목 있음)
- 분야별 교육과정 소개 (과정 제목 있음)

## Notes

- pypdfium2 라이브러리 필요 (pip install pypdfium2)
- 이미지 스케일 기본값 2.0 (고해상도)
- Sub Agent 방식으로 /clear 없이 자동 처리
- 교육과정 제목만 있어도 추출 (상세 정보 없어도 OK)
